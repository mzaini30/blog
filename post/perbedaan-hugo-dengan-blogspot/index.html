Perbedaan Hugo Dengan Blogspot

September 29, 2018

Sebenarnya sangat nggak relevan atau nggak adil banget ya membandingkan Blogspot dengan Hugo. Secara, sistem di antara keduanya sudah sangat berbeda jauh. Ya, mungkin di sini nggak cuma menampilkan perbedaannya tapi juga persamaannya. Jadi intinya, selamat membaca ya.

  1. Sama-sama Blog

    Oke, keduanya sama-sama blog. Mau pakai Blogspot ataupun Hugo, kalau kamu nggak ngisi itu blog ya nggak jalan-jalan lah.

  2. Kalau Hugo Menggunakan Sistem di OS Buat Menjalankan Sedangkan Blogspot, Kita Hanya Perlu Browser

    Kalau Hugo ini bagusnya dijalankan di linux nggak tau ya kalau di windows bisa apa nggak. Nah, kalau Blogspot kan, yang penting kita punya Firefox aja atau Chrome, maka kita sudah bisa menggunakannya.

  3. Hugo Bisa Dijalankan Offline

    Cukup dengan mengetikkan perintah Hugo server di Terminal maka kita sudah bisa menjalankan Hugo. Bayangin lah kalau misalnya kamu berada di tempat yang sangat terpencil dan kamu pengeeen banget buat blogging. Kan susah tuh. Nunggu loadingnya lamaaaa banget kalau Blogspot karena sinyal cuma ada Edge. Kalau Hugo kan bisa aja uploadnya ntar-ntar.


  4. Hugo Pakai Markdown Buat Nulis Sedangkan Blogspot Sudah WYSIWYG

    Kalau di Blogspot, kamu tinggal nulis macam di Word aja. Kalau Hugo, pakai Markdown.

    Sebenarnya Markdown ini mudah kah susah sih? Ya, tergantung kamunya aja mandangnya gimana. Kalau menurutku sih lebih gampang, lebih fleksibel, dan lebih efisien. Contohnya aja di bawah ini:

    1. Tebal

      **tebal**

    2. Miring

      _miring_

    3. Link

      [teks link](http://ini-linknya.com)

    4. Gambar

      ![keterangan gambar](link-menuju-gambar.jpg)

    5. Paragraf

      Ini isi paragraf satu.

Ini isi paragraf dua.

    6. Tabel

      | judul | judul |
|---|---|
| isi | isi |

  5. Perlu Sedikit Kemampuan Pemrograman untuk Menjalankan Hugo

    Kalau untuk edit template di Blogspot bisa dibilang mudah lah karena kita hanya harus paham sedikit dari HTML, Javascript, dan CSS. Dan memang ketiganya itu logikanya mudah dipahami. Berbeda dengan Hugo yang menggunakan Go Templating. Agak aneh sih bahasanya.

    Misalnya saja:

    Javascript Go Templating
    if (argumen_1 && argumen_2){} {{ if and (argumen_1) (argumen_2) }} {{ end }}
    if (!argumen){} {{ if (ne argumen) }} {{ end }}
    var = 'nilai' $var := 'nilai'
    for (n in list){} {{ range list }} {{ end }}

  6. Kalau Ngedit Postingan di Hugo, Dia Reload Sendiri Habis Kita Edit

    Ini yang membuat aku kagum lah. Nggak ada di Blogspot kayak gini. Bahkan di tools tantangannya, yaitu Jekyll juga nggak ada. Mantap wal.
