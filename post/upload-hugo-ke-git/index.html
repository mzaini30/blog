Upload Hugo Ke Git

September 26, 2018

Clone Repositori

Pertama, kamu buat dulu dua repositori. Yang pertama, bertugas sebagai penampung source code, sedangkan yang satunya bertugas sebagai penampil hasil compilenya. Kalau aku, yang sourcenya tak taro di Gitlab sedangkan yang tampilnya di Github yaitu di blog ini -> http://blog2.muhammadzaini.com.

Anggap aja dua repositori ini:

  • source-blog (ini di Gitlab)
  • blog (ini di Github Pages)

Kemudian clone tuh dua repositori itu.


Buat Hugo

Ketikkan perintah ini di Terminal:

hugo new site nama-blognya

Maka, akan jadi folder dengan nama nama-blognya/.

Kemudian, compile Hugo dengan perintah hugo. Maka, setelah itu akan terbentuk folder public/.


Hubungkan Hugo ke Gitlab dan Github Pages

Buka folder source-blog/ kemudian ketikkan Ctrl H. Maka, akan tampil semua folder dan file yang hidden. Copy folder .git/ ke nama-blognya/.

Lalu, buat file dengan nama .gitignore. Ini isinya:

public/

Itu fungsinya supaya semua yang di folder public/ nggak terupload ke repositori yang Gitlab.

Kemudian, buka folder blog/ kemudian copy folder .git/ ke nama-blognya/public/


Skrip Lagi

Beri nama kode di bawah ini dengan upload-blog dan simpan di folder Path.

upload_git(){
	git status
	git add -A .
	git commit -m 'oke'
	git push
}

while true
	do

		# upload dulu ke source blog

		upload_git

		# deploy

		hugo

		# masuk ke folder public

		cd public

		# upload lagi

		upload_git

		# keluar dari folder

		cd ..

		# untuk pause

		read -rsp $'Tekan sembarang tombol untuk upload lagi.\n\n\t'"blog"$'\n\n' -n 1 key

	done

Cara Memanggilnya

Pada folder blog Hugo, jalankan aja upload-blog.
